<h1> Cuber </h1>
An infinite runner game
My first ever game which helped me learning game development
<h2>Controls</h2>
<- (left arrow key):- To go left<br>
-> (right arrow key):- to go right<br>
(spacebar):- to jump

<h2> Features </h2>
<ul>
<li>Random pattern of obstacles
<li>Maintains High score
</ul>
